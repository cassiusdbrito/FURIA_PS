# Chat FURIA

Interface conversacional para os fãs da FURIA.

## Como rodar o projeto

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Acesse em: http://127.0.0.1:8000/

## Tecnologias usadas
- Django
- HTML/CSS/JS
